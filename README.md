Comando: git --version
Definición: Muestra la versión instalada de Git.
Ejemplo: git --version

Comando: git init
Definición: Inicializa un repositorio Git en la carpeta actual.
Ejemplo: git init

Comando: git add
Definición: Agrega archivos o cambios al área de preparación (staging).
Ejemplo: git add .   (agrega todos los archivos)

Comando: git commit
Definición: Registra los cambios en el repositorio con un mensaje.
Ejemplo: git commit -m "Primer commit"

Comando: git status
Definición: Muestra el estado actual del repositorio (archivos sin seguimiento, modificados, etc.).
Ejemplo: git status
